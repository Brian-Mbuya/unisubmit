ABSTRACT
The rapid growth of urban populations has led to significant traffic congestion in major cities. This project proposes an IoT-based deep learning system designed to predict traffic volume and optimize routing decisions in real time. We utilize street-level IoT sensor arrays to gather speed, occupancy, and vehicle count data.

PROBLEM STATEMENT
Traditional traffic management systems rely on static timer models and historical data, which fail to adapt to dynamic road conditions, leading to excessive delays, greenhouse gas emissions, and economic losses. There is a critical lack of decentralized, real-time prediction frameworks that can forecast congestion before it occurs and dynamically suggest alternative pathways.

OBJECTIVES
1. Deploy real-time street sensor simulation streams using MQTT.
2. Develop a Long Short-Term Memory (LSTM) recurrent neural network model to predict congestion.
3. Construct a dynamic routing API using Dijkstra's algorithm optimized with prediction scores.
4. Implement a responsive web interface for commuter alerts and traffic authority visualization.

TECHNOLOGIES
The system will be implemented using Python for data preprocessing and LSTM model training, utilizing TensorFlow and Keras. The backend infrastructure will be built on Spring Boot and Java, communicating with H2 Database for local relational storage. MQTT broker (Eclipse Mosquitto) will handle sensor streams, and the frontend will use React with TailwindCSS.

RESEARCH AREAS
The research falls under the domains of Smart Cities, Distributed Systems, and Deep Learning.
