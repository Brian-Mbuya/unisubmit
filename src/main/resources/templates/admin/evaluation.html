How this is measured

0 pairs

Every accepted collaboration request is treated as proof that the requested project was genuinely relevant to that student. For each weight configuration the engine re-ranks each student's candidate pool live and records where the accepted project actually landed. Precision@5 is the share of pairs ranked in the top five; MRR (mean reciprocal rank) rewards ranking relevant work near the top. Nothing here is persisted — it is a what-if replay over real usage data.
